# Chairtrade
you trade chairs to get to better chairs

## Trade guide
3 Euc --> 1 Vuc

5 Vuc --> 1 Uc

3 Uc --> 1 Sc

5 Uc --> 1 Spuc

3 Spuc --> 1 Quc

10 Spuc --> 1 C

3 C --> 1 B
